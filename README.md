# scnvim-arm

[SuperCollider][supercollider] integration for [Neovim][neovim] on ARM devices

## Note

Tested on Arch-Arm

This branch is basically archived - the master branch (and of course upstream/master) now supports using other help file creators than pandoc.
So vanilla `scnvim` is good to go on ARM devices. <3

Thanks to David for the wonderful `scnvim`

[supercollider]: https://github.com/supercollider/supercollider
